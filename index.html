Lesson 1: The Basics of the Web and HTML.

World Wide Web

World Wide Wed is a collection of HyperText Markup Language (HTML). HTML is the heart of the web. It is the main type of document on the web. HTTP is the main protocol of the web. Major pieces of web are servers, internet and browsers ( like IE, Google Chrome and Firefox)

Intro to HTML

HTLM markups are composed of tags. Examples are bold tags and emphasis tag. An example of tags with HTML attributes are anchor tag. For now we dont have to memorize all the tags, a Google search for "adding links in html" will always get you your answer.

Inline vs Block Elements

HTML elements are either inline or block. Example of Inline elements are a, span, img, strong, and br. Block elements form an "invisible box", examples are p, div and form.
